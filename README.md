## What I'm Building

### SmartCharge — EV Charging SaaS Platform
A cloud-native platform for public and fleet EV charging.
- Architected an OCPP/OCPI gateway orchestrating ~1,000 public & fleet chargers
- Built a real-time billing engine (pay-as-you-go, subscription, fleet settlement) processing **$1M+** in annual transactions
- Introduced Kafka-based event streaming & edge caching to cut charge-session latency
- Implemented ISO-15118 "Plug-and-Charge" plus dynamic load-balancing

🔗 [smartcharge.app](https://smartcharge.app)

### EV Travel Diary & Planner
A cross-platform (iOS/Android) travel diary and trip planner for EV drivers.
- Integrated charging availability, route optimization, and vehicle telemetry

### What I Eat — AI Nutrition Tracker
An AI-powered nutrition app built with SwiftUI (MVVM-C).
- Real-time food scanning via camera and voice input
- AI Nutritionist chat with diet-tailored responses, body-composition analysis, and gamified rewards
- StoreKit subscriptions, HealthKit sync, localization, and a custom Spring Boot backend with full CI/CD

🔗 [Available on the App Store](https://apps.apple.com/us/app/what-i-eat-track-food-earn/id6756741842)
